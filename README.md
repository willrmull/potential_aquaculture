# Indentifying Areas of Potential Aquaculture in the Western US

**Author** William Mullins

### About

This repository contains a quarto markdown document of my analysis on areas suitable for aquaculture in the US. The analysis looked into regions which had suitable temperature and depth for the farming of either oysters or blue mussels.
